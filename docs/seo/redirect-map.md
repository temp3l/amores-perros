# Weiterleitungsplan

## Alte Export-URLs

| Alte URL | Neues Ziel | Hinweis |
| --- | --- | --- |
| `/index.html` | `/` | Kanonische Startseite |
| `/home-1.html` | `/` oder finale Unterseiten | Duplikat der Startseite |
| `/contact-us.html` | `/kontakt/` | Alte Kontaktseite |
| `/about.html` | `/ueber-jacky-rebien/` | Über-mich-Seite |
| `/informationen.html` | `/impressum/` | Gesetzliche Seite |
| `/https/amores-perrosde/abouthtml/über.html` | `/ueber-jacky-rebien/` | Export-Artefakt |

## Regeln

- Duplikate immer auf ein einziges kanonisches Ziel leiten.
- Keine mehrfach indexierbaren Seiten mit gleichem Inhalt behalten.
- Alte gemischte Absolute-URLs nach dem Launch entfernen.
