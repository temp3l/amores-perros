# Rebuild Sequence

## Ziel

Die Website wird in einer festen Reihenfolge aufgebaut, damit zuerst die geschäftskritischen Seiten und danach die Ergänzungen umgesetzt werden.

## Phase 1: Must-have

1. Startseite
2. Erstgespräch
3. Einzeltraining
4. Ueber mich
5. Kontakt
6. Preise
7. Impressum
8. Datenschutz

## Phase 2: Core Expansion

1. DOGSpace
2. Workshops und Seminare
3. Coaching mit Hund
4. FAQ
5. Trust- und Credential-Module

## Phase 3: Optional / später

1. Hundetrainer-Austausch
2. Supervision
3. Mantrailing-Training
4. Event-Archive oder Terminlisten, falls wirklich regelmaessig Termine gepflegt werden

## Umsetzungsregeln

- Keine Seite wird ohne klare Ziel-URL und eindeutigen Suchintent gebaut.
- Preise, Zeiten und Kontaktinfos kommen ausschliesslich aus der Business-Faktenbasis.
- Sekundaere Angebote bleiben getrennt von den Kernangeboten.
- Alte Export-Artefakte werden per Redirect abgefangen, nicht weiterverwendet.
